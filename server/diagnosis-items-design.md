## 점수 산정 방식

1. 각 항목의 원점수(rawScore) 산출 (기존과 동일)
2. 카테고리별 원점수 합산
3. 진료과 가중치 적용: 카테고리점수 x 가중치
4. 전체 합산 후 100점 만점으로 환산
5. 진료과 미지정 시 기본값(1.0) 적용 = 기존과 동일한 결과
