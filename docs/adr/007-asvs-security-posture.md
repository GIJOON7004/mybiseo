# ADR-007: OWASP ASVS 보안 수준 결정

**상태:** 승인 (2026-05-10)
**작성자:** Manus AI

---

## 맥락

3개 외부 AI 코드리뷰(제미나이, 젠스파크, 클로드)에서 OWASP ASVS Level 3 적용을 제안했습니다. ASVS Level 3은 금융, 군사, 의료 EMR 등 고위험 시스템을 대상으로 286개 보안 요구사항을 정의합니다. 마이비서의 보안 수준을 어디까지 적용할지 결정이 필요합니다.

## 결정

**ASVS Level 2를 기본 적용하되, Level 3에서 V7(Logging), V8(Data Protection), V14(Secret Management) 3개 챕터만 선별 적용합니다.** Level 3 전체 286개 요구사항은 적용하지 않습니다.

## 근거

### 마이비서가 다루는 데이터의 민감도

마이비서는 환자 PHI(Protected Health Information)를 직접 저장하지 않는 SEO 진단 SaaS입니다. 그러나 다음과 같은 민감 데이터를 다룹니다:

| 데이터 유형 | 예시 | 법적 분류 |
|------------|------|----------|
| 병원장 개인정보 | 이메일, 전화번호, 이름 | 개인정보보호법상 개인정보 |
| 병원 경영정보 | 매출 추정치, 경쟁사 분석 | 영업비밀 |
| 병원 메타데이터 | 병원명, 진료과목, 의료진 정보 | 공개 정보 (단 집합 시 민감) |

환자 PHI 미저장으로 HIPAA 직접 적용 대상은 아니지만, 개인정보보호법과 영업비밀보호법의 적용을 받습니다. 따라서 Level 1~2만으로는 부족하고, Level 3의 핵심 챕터를 선별 적용하는 것이 적절합니다.

### Level 3 전체 미적용 사유

ASVS Level 3의 286개 요구사항 중 상당수는 EMR, 금융 거래, 군사 시스템에 특화된 항목(하드웨어 보안 모듈, 물리적 보안 등)으로, B2B SEO SaaS에는 과잉입니다. 현재 비즈니스 모델과 팀 규모(1인)를 고려할 때 전체 적용의 ROI가 부족합니다.

## V7 Logging Audit 결과

| 요구사항 | 현재 상태 | 판정 |
|----------|----------|------|
| V7.1.1: 모든 인증 이벤트 로깅 | OAuth 콜백에서 로깅 (server/_core/oauth.ts) | **통과** |
| V7.1.2: 모든 접근 제어 실패 로깅 | protectedProcedure에서 UNAUTHORIZED 에러 발생 시 tRPC 에러 로그 | **통과** |
| V7.1.3: 입력 검증 실패 로깅 | Zod 스키마 검증 실패 시 tRPC BAD_REQUEST 에러 로그 | **통과** |
| V7.2.1: 로그에 민감정보 미포함 | logger.ts에서 구조화된 로깅, PII 필드 미포함 | **통과** |
| V7.3.1: 로그 무결성 보호 | stdout → Manus Cloud Logging 자동 수집 (변조 불가) | **통과** |
| V7.4.1: 감사 로그(audit trail) | **미구현** — 사용자 행위 감사 로그 테이블 없음 | **미통과** |

**조치 필요:** V7.4.1 감사 로그는 현재 미구현이나, 인수실사 시점에서 필요할 수 있습니다. 재검토 트리거에 포함합니다.

## 재검토 트리거

다음 조건 중 하나라도 충족되면 ASVS 수준을 재평가합니다:

1. **환자 데이터 저장 기능 추가** — Level 3 전체 적용 필수
2. **팀 규모 3인 이상** — 감사 로그(V7.4.1) + 시크릿 로테이션(V14.2.1) 즉시 구현
3. **인수실사 진행** — 미통과 4개 항목 사전 해결
4. **개인정보 유출 사고 발생** — 전체 보안 재감사

## 대안 (기각)

- **Level 3 전체 적용:** 286개 요구사항 중 50% 이상이 현재 서비스에 해당 없음. 과잉 투자.
- **Level 1만 적용:** 병원장 PII를 다루는 서비스에 Level 1은 부족. 개인정보보호법 위반 리스크.
- **SOC 2 Type II:** 연간 $20K+ 비용. 현재 매출 규모에서 ROI 부족. 인수실사 시점에 재검토.
